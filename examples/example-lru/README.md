# example-lru

Example of reducing memory usage by fetching from the file (with LRU cache) in jpreprocess.

Note: This example only works with JPreprocess dictionary (not lindera dictionary).

## Usage

Usage is the same as example-simple, except that `./tests/data/dict` needs to be jpreprocess dictionary (In `tests` crate README, set up the full dictionary with "JPreprocess dictionary" procedure).
